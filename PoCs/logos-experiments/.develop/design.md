# Design Spec — vjepa_clip_alignment

## Architecture

**Residual MLP Projector**
- Input: (N, 1024) mean-pooled V-JEPA embeddings
- Layer 1: Linear(1024 → 1024)
- Trunk: 4× Residual blocks: `x + Linear(1024→1024) → GELU → LayerNorm(1024)` + Dropout(0.1)
- Output: Linear(1024 → 768) → L2-normalize per row
- NaN/Inf safety: LayerNorm stabilises; L2-norm enforced at output

## Loss Schedule

- Epochs 1–30 (warmup): MSE against mean-pooled clip_text targets
- Epochs 31+: InfoNCE (symmetric, temperature=0.07)

## Training Config

- AdamW: lr=4.5e-5, weight_decay=0.05
- Cosine LR schedule
- grad_clip=0.4
- batch_size=128
- NaN/Inf detection: stop if 10 consecutive bad batches
- Best checkpoint → `experiments/vjepa_clip_alignment/best.pt`
- Data split: 80/20 (train 5608 / val 1402)

## Subtask Dependency Graph

```
T1: data_prep ──┐
                ├──► T3: train ──► best.pt ──► full eval
T2: model    ───┘
T2: model    ───────► T4: projector ──► smoke test
```

T1 and T2 are independent (run in parallel).
T4 can be smoke-tested before T3 finishes (uses random weights).

## Task Details

### T1 — data_prep.py
- Blocks: T3
- Done: `data/val/jepa_val.npy` (7010,1024) and `data/val/clip_val.npy` (7010,768) exist, all-finite
- Key: jepa h5['jepa'].mean(axis=1), clip h5['clip_text'].mean(axis=1)

### T2 — model.py
- Blocks: T3, T4
- Done: `ResidualMLP()(torch.randn(16,1024))` → (16,768), row norms ≈ 1.0, no NaN
- Exports: ResidualBlock, ResidualMLP, save_checkpoint, load_checkpoint

### T3 — train.py
- Depends: T1, T2
- Done: training completes without NaN, best.pt saved, training_stability==1.0
- Exports: infonce_loss, train(); runnable as CLI

### T4 — projector.py
- Depends: T2
- Done: `python eval/test_alignment.py --projector workspace/projector.py --synthetic` exits 0
- Exports: load(checkpoint_path=None) -> state, project(state, vjepa_embs) -> np.ndarray

## Test Strategy

| Task | Test | Pass condition |
|------|------|----------------|
| T2 | Unit: random (16,1024) forward | shape==(16,768), norms≈1.0, no NaN |
| T1 | Integration: run extract, check files | shapes correct, np.isfinite().all() |
| T4 | Smoke: eval --synthetic | exit 0 |
| T3 | End-to-end: run train.py | val_cosine_sim logged, no NaN |
| All | eval with best.pt | R@5≥0.80, R@1≥0.50, cosine≥0.82, stability==1.0 |

## Implementation Order

1. T1 + T2 in parallel

3. T3 once T1 + T2 done
